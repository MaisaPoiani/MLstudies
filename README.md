# Studies in Machine Learning 
![cover image](/machineLearning.png) 

This repository contains a collection of practical exercises and case studies developed while exploring Machine Learning fundamentals. The projects focus on model implementation, data preprocessing, and critical analysis of model performance and ethics.

# 📂 Repository Structure
The core notebooks are located in the ml/cc/exercises/ directory and cover the following topics:

### Modeling Exercises
#### linear_regression.ipynb: 
Implementation of linear regression models to predict continuous values.
#### binary_classification.ipynb:
Development of a binary classifier to sort types of grains, exploring metrics such as precision, recall, and the ROC curve.
#### fairness_income.ipynb: 
A study on Fairness in AI, analyzing income prediction models for bias and implementing techniques to ensure equitable model outcomes.

### Data Wrangling & Analysis
#### numerical_data_outliers.ipynb:
Techniques for detecting and visualizing outliers in numerical datasets using plotting methods.
#### numerical_data_bad_values.ipynb: 
Strategies for identifying and handling inconsistent data or "bad values" that can skew model results.

### 🛠️ Technologies Used
- Python 3
- Pandas & NumPy: For data manipulation and matrix operations.
- Matplotlib: For data visualization and outlier analysis.
- Google Colab: Primary development environment.

### 🚀 How to Use
Clone the repository:
```
git clone https://github.com/MaisaPoiani/MLstudies.git
```

Navigate to the exercises folder:
```
cd MLstudies/ml/cc/exercises/
```

Run the notebooks:
Open the files using Jupyter Notebook, VS Code, or upload them directly to Google Colab.

## 📝 Learning Objectives
Each notebook includes comments on preprocessing decisions, hyperparameter tuning, and result interpretation, with a specific focus on mitigating bias and handling real-world numerical data challenges.
